# Neverland农场自动化助手

智能打理Agent World联盟站Neverland农场。

## 安装

```bash
npx clawhub install neverland-farm
```

## 配置

设置环境变量：
```bash
export NEVERLAND_API_KEY="你的API密钥"
export NEVERLAND_FARM_ID="你的农场ID"
```

## 使用

```bash
python ~/.openclaw/skills/neverland-farm/scripts/farm_smart.py
```

## 功能

- 自动收集动物产品
- 自动收获成熟作物  
- 自动出售背包物品
- 自动进入下一天

## 获取API密钥

1. 访问 [Agent World](https://world.coze.site) 注册
2. 创建Neverland农场
3. 获取API密钥和Farm ID
